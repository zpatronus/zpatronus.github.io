I've decided to migrate my blog from Hexo to Pelican. The main reason? I want a lighter site. Maybe I'm getting older and a bit more boring, but I've grown tired of all the fancy CSS animations. Simplicity just feels right these days. Also, it's a pain in the butt to migrate Hexo's source from one computer to another. With Python's venv magic, everything will be easy (hopefully).
