God of Avalon

Date: Tue 03 May 2022
Updated: Tue 03 May 2022

In 1. Projects.

Tags: Project Fun Stuff Web Dev Django Vue

This is a online multiplayer real-time platform for the board game The Resistance: Avalon.

Rules of the game

For users

If you are in SJTU campus/using an SJTU vpn (as long as you has an SJTU ip), you can visit this site to enjoy the online Avalon god.

~~I haven't deployed it to any VPS, but I do have the plan.~~

Update: it has been deployed to a VPS. Visit this site to enjoy it.

The interface should be straight forward for people who have played this game before.

The polling interval is set to 4 seconds, so please a moment if you notice some desynchronization.

For developers

This project is built with Vue+Django.
